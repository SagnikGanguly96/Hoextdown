# Hoextdown
Hoextdown is an extension to Hoedown, originally located at https://github.com/kjdev/hoextdown. This is a port of the Hoextdown for Windows.


How to Build
================================
## *Requirements*

|	Platform Toolset		|	Windows SDK		|	Architecture	|
